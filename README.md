# phog
Tool for printing PHP variables in navigator javascript console.


```php
require "vendor/autoload.php";
$phog = new \Novia713\Phog\Phog();
$phog->log($value);
$phog->info($arr);
$phog->warn("¡¡autodidacta en la sala!!");
$phog->table($obj);
```
